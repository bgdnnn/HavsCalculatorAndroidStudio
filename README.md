# HAVS Calculator (Android)

Offline HAVS calculator using the UK HSE points system.

## Features
- Tool database (maker/model, vibration m/sÂ², optional noise dB, optional max minutes to 350)
- Calculate points from minutes
- Points per minute display
- Daily history saved (resets automatically each new day)
- Edit / delete tools
- Cloud background UI

## Tech
- Kotlin + Jetpack Compose (Material 3)
- DataStore (JSON)
- MVVM
